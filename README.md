# GoBus Transportation Booking System

A modern full-stack transportation booking platform that allows passengers to search trips, book seats, generate QR-code tickets, and manage bookings through a professional administration dashboard.

---

##  Tech Stack

### Backend

* Laravel 12
* Laravel Sanctum
* MySQL
* REST API
* Eloquent ORM


✔ Queue

✔ Scheduler

✔ Notifications



### Frontend

* React.js 19
* Vite  8
* Tailwind CSS
* React Router
* Axios
* React Icons
* React Toastify
* React Hook Form



Deployment

✔ Docker
✔ CapRover
✔ Nginx
✔ PHP-FPM
✔ Supervisor
✔ Redis
✔ MariaDB
---

##  Features

### Passenger Features

* User Registration & Login
* Search Trips
* View Trip Details
* Book Seats
* Online Ticket Generation
* QR Code Verification
* My Trips Management
* User Profile Management
* Notifications System

### Admin Features

* Dashboard Statistics
* Users Management
* Cities Management
* Stations Management
* Buses Management
* Trips Management
* Bookings Management
* Reviews Management
* Activity Logs

---

## Dashboard Statistics

The admin dashboard provides:

* Total Users
* Total Trips
* Total Bookings
* Total Tickets
* Total Revenue
* Today's Revenue
* Upcoming Trips Table

##  Authentication

Authentication is implemented using:

* Laravel Sanctum
* Protected API Routes
* Admin Middleware
* Role-Based Access Control

---



## ⚙️ Installation

### Backend

```bash
cd gobus-api

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate --seed

php artisan serve
```

### Frontend

```bash
cd gobus-frontend

npm install

npm run dev
```
